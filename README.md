## 🇻🇳 Tiếng Việt (VIE)

### [28/5/2026] Một số vấn đề nhỏ

1. Cấu trúc file ngôn ngữ của mình được dựa trên file `Example.txt` trên iOS. Vì vậy nếu bạn cài file ngôn ngữ này trên Android, có thể game sẽ hiện thông báo như:
   - `Your custom translation is out of date (English will be used for missing translations)`
   - hoặc `Missing:` và `Unnecessary:`

   Đừng lo hoặc nghĩ rằng file bị lỗi. Đây chỉ là do sự khác nhau về cấu trúc file ngôn ngữ giữa Android và iOS thôi. Trên Android, bạn chỉ cần nhấn `Bỏ qua/Đóng` ở phía dưới là được, game vẫn sẽ nhận đầy đủ và hoạt động bình thường.  
   Trên iOS sẽ không xuất hiện các thông báo này.

2. File ngôn ngữ tiếng Thái hiện tại vẫn đang ở giai đoạn `Beta`, nên có thể vẫn còn lỗi hoặc một vài chỗ mình để sót. Nếu phát hiện lỗi, hãy báo lại giúp mình để mình có thể cải thiện bản dịch nhanh nhất có thể.

3. File tiếng Thái trên iOS hiện đang bị lỗi text, bản vá sẽ sớm được phát hành.

## 🇬🇧 English (ENG)

### [28/5/2026] Some Important Notes

1. The structure of this language file is based on the `Example.txt` file from iOS. Because of that, if you install this translation on Android, the game may display messages such as:
   - `Your custom translation is out of date (English will be used for missing translations)`
   - or `Missing:` and `Unnecessary:`

   Don't worry or assume that the file is broken. This only happens because the language file structure differs between Android and iOS. On Android, you can simply press the `Ignore/Close` button at the bottom, and the game will still load and work normally with the full translation applied.  
   On iOS, these warnings should not appear.

2. The Thai language file is currently still in `Beta`, so there may still be bugs, untranslated lines, or overlooked mistakes. If you find any issues, please report them so I can improve the translation as quickly as possible.

3. The Thai language file on iOS is currently experiencing text issues. A patch will be released soon to fix them.
