# นโยบายด้านความปลอดภัย

## เวอร์ชันที่รองรับ

ด้านล่างคือสถานะการรองรับการอัปเดตด้านความปลอดภัยสำหรับโปรเจกต์แปลภาษานี้

| Version | Supported |
| ------- | ---------- |
| Beta    |  Yes       |
|         |            |

## การรายงานช่องโหว่

หากคุณพบปัญหาด้านความปลอดภัย ช่องโหว่ หรือข้อผิดพลาดร้ายแรงภายในไฟล์คอนฟิกที่อาจส่งผลต่อเสถียรภาพของเกม โปรดรายงานโดยตรงผ่าน Discord

### ขั้นตอน

- **ช่องทางติดต่อ:** ส่งข้อความส่วนตัว (DM) ผ่าน Discord ถึง `@pongb`
- **ระยะเวลาตอบกลับ:** คุณจะได้รับการตอบรับหรืออัปเดตเบื้องต้นเกี่ยวกับปัญหาที่รายงานภายใน 72 ชั่วโมง
- **การแก้ไข:** หากยืนยันว่าช่องโหว่หรือข้อผิดพลาดของไฟล์มีอยู่จริง จะมีการปล่อย hotfix หรือเวอร์ชันปรับปรุง (เช่น v0.x) ไปยัง repository ทันที
- **การแจ้งเตือน:** ผ่าน [Forum](https://sfsforum.com) หรือผ่าน releases
