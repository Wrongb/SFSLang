# Spaceflight Simulator - Thai Localization Update Log

**Version:** Beta  
**Translator:** Pongb  

## English (Changelog)

* **Initial Beta Release:** First playable beta version of the Thai language pack.
* **Translation Cleanup:** Completely scrubbed and corrected broken or corrupted characters from the raw draft.
* **Terminology Standardization:** 
    * Kept familiar English gaming terms for better player experience: *Blueprint, Sandbox, Career, Mod, Prograde, Retrograde*.
    * Standardized specific Thai terms: *Cheats -> สูตรโกง, Quicksave -> เซฟด่วน, Stage -> สเตจ, Part -> ชิ้นส่วน*.
* **UI/UX Optimization:** Shortened and condensed text strings to prevent UI overflow on mobile screens (e.g., shortened "Launch Anyway" to "ปล่อยเลย", and optimized the Code Mod mobile warning).
* **Tone Polish:** Refined the overall sentence structures to feel more natural, casual, and closely aligned.

## ภาษาอังกฤษ (บันทึกการเปลี่ยนแปลง)
* **เวอร์ชันเบต้าเริ่มต้น:** เวอร์ชันเบต้าที่เล่นได้จริงเวอร์ชันแรกของชุดภาษาไทย
* **การปรับปรุงการแปล:** ตรวจสอบและแก้ไขตัวอักษรที่เสียหายหรือผิดพลาดจากฉบับร่างต้นฉบับอย่างสมบูรณ์
* **การกำหนดมาตรฐานคำศัพท์:**
* คงไว้ซึ่งคำศัพท์เกมภาษาอังกฤษที่คุ้นเคยเพื่อประสบการณ์การเล่นที่ดีขึ้น: *Blueprint, Sandbox, Career, Mod, Prograde, Retrograde*
* กำหนดมาตรฐานคำศัพท์ภาษาไทยเฉพาะ: *Cheats -> สูตรโกง, Quicksave -> เซฟด่วน, Stage -> สเตจ, Part -> ส่วน*
* **การปรับปรุง UI/UX:** ย่อและกระชับข้อความเพื่อป้องกัน UI ล้นบนหน้าจอมือถือ (เช่น ย่อ "Launch Anyway" เป็น "ปล่อยเลย" และปรับปรุงคำเตือน Code Mod บนมือถือ)
* **การปรับแต่งน้ำเสียง:** ปรับปรุงโครงสร้างประโยคโดยรวมให้รู้สึกเป็นธรรมชาติ เป็นกันเอง และสอดคล้องกันมากขึ้น
